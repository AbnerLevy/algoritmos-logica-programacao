# 💻 Algoritmos e Lógica de Programação

Bem-vindo ao meu repositório de estudos de programação! Este espaço é dedicado ao registro da minha evolução técnica, contendo exercícios, projetos e algoritmos desenvolvidos durante a minha graduação em **Análise e Desenvolvimento de Sistemas (ADS)**[cite: 1].

## 📌 Sobre o Repositório
O objetivo deste repositório é servir como um histórico de aprendizado e portfólio técnico. Aqui você encontrará:
* **Exercícios de Fixação:** Problemas de lógica resolvidos em sala de aula.
* **Projetos Acadêmicos:** Aplicações completas que integram diversos conceitos fundamentais.
* **Desafios Práticos:** Implementações que exploram o máximo das ferramentas disponíveis em cada etapa do curso.

## 📂 Estrutura de Pastas
Para facilitar a navegação, o repositório está organizado por semestres:
* **`semestre-1/`**: Focado na base de algoritmos e linguagem C.
  * `desafio-reino/`: Um RPG de texto completo explorando lógica estruturada, operadores e laços de repetição.

## 🛠️ Tecnologias e Ferramentas
* **Linguagens:** C (até o momento)
* **Ambiente:** VS Code e compilador GCC (MinGW).[cite: 1]
